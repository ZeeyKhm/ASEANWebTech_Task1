# ASEANWebTech_Task1
Graphical interface design task

A customer asked to build a marketing website for a legal company that helps businesses and individuals get water-use permits across Russia. The site should explain the services, show past work, answer common questions, and collect client inquiries (consultation requests and rent calculation forms).

What the site should include:

- Header: ogo, navigation links (About, Services, Cases), address, phone, social/messenger links.
- Hero section: big image, main title about water-use agreements, subtitle about nationwide coverage, button to scroll down.
- Services section: list of service categories with short descriptions and a button to contact via messenger.
- How we work: 5-step process, from free consultation to document registration.
- About company: description, leader’s photo/name/role, key numbers (projects per year, approvals, active clients), and a short video.
- Projects (cases): scrollable list of project cards with location, work type, document type, and image(s). Click to open image gallery if multiple images exist.
- Testimonials: auto-scrolling carousel of client reviews with 5-star ratings; pause on hover.
- Rent calculator block:
  - Fields: length (m), width (m), term (years).
  - Show success or error message. Clear form on success.
  - Include links to privacy policy and consent documents.
- FAQ: expandable list of legal Q&A, followed by a **consultation form** (name, phone, email).
- Footer: same logo, links, and contacts as header.

The client also clarified that they don't have any pre-written text for the page. All text is left to the designer's discretion. However, it must be consistent with the company's theme. The same applies to images.

REQUIREMENTS FOR THE RESULT
The student must attach a link to Figma with the generated design and provide a short report on the work done: which AI were used, in what order, and what prompts.

It's recommended to use AI in order to achieve efficient result.

1. Google Stitch (https://stitch.withgoogle.com) - a great tool for prototyping design for websites

2. Cursor (https://cursor.com) - a code editor with AI agents

3. Google Whisk (https://labs.google/fx/tools/whisk) - an image generator

4. skillsmp (https://skillsmp.com) - a list of agent skills

5. perplexity (https://perplexity.ai) - an AI-powered search engine

6. ChatGPT (https://chatgpt.com) - an AI chatbot to create content
